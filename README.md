# Amerimax Digital Catalog Engine

### **Project Mission**
Re-engineering the **Amerimax for Mobility Digital Parts Catalog** to bridge the gap between complex industrial inventory and user-centric web design. This project focuses on transforming a high-volume product list into a high-performance, accessible digital tool for contractors.

---

### **Engineering Challenges & Solutions**
* **The Technical Gauge Conflict:** Solved the confusion between material thickness (e.g., **0.010 gauge**) and manufacturer color codes (e.g., **010 Polar White**) through a visually isolated UI architecture.
* **Cognitive Ease for Contractors:** Designed a **"Quick-ID"** interaction system using a rotating plus-sign ($+$) accordion. This provides a 5-word micro-definition for every part, reducing ordering errors and support calls.
* **High-Contrast Accessibility:** Custom CSS overrides built specifically for field use on mobile devices, ensuring legibility for an older demographic of professionals.

---

### **Core Information Architecture**
The catalog is optimized into 8 primary industrial categories:
1. **Fascia** – Roof edge protection
2. **Soffit** – Under-roof ventilation
3. **Mini Gutter** – Specialized RV drainage
4. **Flashing** – Joint leak prevention
5. **Starter Strip** – Base locking systems
6. **Corner** – Wall edge sealing
7. **Verticals** – Upright wall protection
8. **Cleated Roofs** – Interlocking high-wind systems
